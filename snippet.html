The Explanation

This snippet uses multiple tags and other features.
For an in-depth look at tags in general, you can check out W3 Schools, Don't Fear the Internet, and Interneting is Hard.
On a basic level, tags "nest" within other tags. Think of a Russian Nesting Doll for visual reference. They close around each other, depending on what they do and how important they are.
For instance, the paragraph tag P goes within the body tag, body. Tags start within two carrots, and terminate within the same two carrots with a backslash appended to the front of the tag.
Paragraph tags display all the text within them, allowing for formatting with various other tags. They can also be broken up with br which doesn't need to be terminated.
a is a tag used to define a link. Within the brackets for the first a you have the attribute href= with the link following in quotation marks.
The first snippet uses three other attributes, id, class, and target. ID is set to "the-best-link" in this case. IDs are used to define distinguish elements and style them.
Next, you have class. In this case, class is set to "banana" Class is used to specify a class for an element, which can be shared. Classes are typically defined in a .CSS document.
Finally, we have target. Target is set to "_blank". Target is used to define where the link opens. In this case, "_blank" opens the link in a new tab.
